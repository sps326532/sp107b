## C 語言版 HackCPU 虛擬機的用法


```
$ gcc asm.c c6.c -o asm
$ ./asm ../add/Add
```

上面的組譯器指令會產生 ../add/Add.bin 檔，於是我們可以用虛擬機執行該機器碼檔案！
